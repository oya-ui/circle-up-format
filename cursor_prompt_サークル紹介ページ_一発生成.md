## そのまま Cursor に貼る指示（英語でも日本語でも可。下記は日本語）

```
あなたはフロントエンド実装担当です。次の要件で、サークル向けの「新歓用・1ページの紹介サイト」を新規作成してください。

## 技術制約
- フレームワークは使わず、静的ファイルのみ（HTML + CSS、必要ならバニラJSのみ）。
- 1リポジトリで完結。エントリは index.html。
- 画像はプレースホルダー（picsum.photos や placehold.co など）でよいが、alt 必須。
- アクセシビリティ：見出し階層（h1→h2→h3）、コントラスト、フォーカス可視、日本語の lang="ja"。
- パフォーマンス：不要なライブラリ禁止。CSS は 1 ファイル（styles.css）にまとめるか、index.html に <style> でよいが、保守しやすい分割を優先。

## デザインの狙い（参考レベル）
参考URLの情報設計に近いこと（見た目の完全コピーではなく、同等の「完成度」）:
https://coruscating-tanuki-7f6bb5.netlify.app/

具体的には次を満たすこと:
- 上部に固定または目立つナビ（ロゴ文字・「について」「活動」「メンバー」「入部」などアンカーリンク）。
- ヒーローセクション：サークル名（h1）、所属大学、短いキャッチコピー、背景に大きなメイン画像（プレースホルダー）。
- 「サークルについて」：2〜3段落の説明 + 箇条書きタグ（例: 初心者歓迎、週1、イベント多数）。
- 「活動内容」：カード3枚（タイトル + 説明文 + 小さな画像枠）。
- 「メンバー/数字」：4つの統計（部員数、設立年、所属大学、初心者率など）をグリッド表示。
- 「幹部/メンバー紹介」：2列またはカードグリッドで、顔写真プレースホルダー + 名前 + 大学/学年 + 一言コメントを最低5名分。
- 「入部・連絡」CTA：見学・体験歓迎の文面 + SNS（Instagram / LINE）のダミーリンク（#可）。
- フッター：コピーライト、サークル名。
```

## フォーム回答を埋め込むときの手順（運用メモ）

1. スプレッドシートの1行目を見て、各列を `{{CIRCLE_NAME}}` 等にマッピングする（手作業または簡単なスクリプト）。
2. 上記プロンプトの `{{...}}` を実データに置換してから Cursor に「この内容でスタイルを微調整して」と依頼するか、初回から実名入りでプロンプトを作る。
