### CSV を取る手順（エージェント向け）

1. 上記 **export** URL に HTTP GET する（`curl` や `WebFetch` 等）。レスポンスは UTF-8 の CSV 1本分。
2. 1行目はヘッダ。**有効データ行の定義（固定）**: **B列＝「正式名称」が空でない行だけ**を集計対象とする（前後の空白はトリムして判定）。A列だけ埋まっているテンプレ行は **含めない**。
3. 列名は次のとおり（`OutreachManagementSheet.gs` と一致。A=団体ID、B=正式名称）:  
   `団体ID`, `正式名称`, `大学`, `エリア`, `ステージ`, `最終接触日`, `次アクション日`, `担当`, `チャネル`, `反応`, `次の一手メモ`

### 別タブを読むとき

シートを追加したら、Google スプレッドシートの URL に含まれる `gid=` を確認し、export URL の `gid=0` を差し替える。`運用_リンク一覧.md` の CSV 行も更新する。

### 取得できないとき

- ログイン画面の HTML が返る、403/401 になる → 共有設定が **リンク知ってる人は閲覧可** になっているか確認。直るまでユーザーに表のコピペを依頼する。
